# stock-frontend

株式データのフロントエンドアプリケーション。

## 技術スタック

- Next.js / React
- TypeScript
- Tailwind CSS

## プロジェクト固有の基礎知識

### 日本株の取引時間（2024年11月～延長後）

- 前場: 9:00-11:30
- 後場: 12:30-**15:30**（大引け）
- **15:30で取引終了**
- 大引け前の手動確認推奨時刻: **15:20頃**（10分の余裕）
- 最終注文リミット: **15:28**

## 開発コマンド

```bash
npm run dev    # 開発サーバー起動
npm run build  # ビルド
```
